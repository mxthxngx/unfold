# Roadmap

## Editor & UX

- Improve click-to-place cursor behavior (Notion-like focus targeting).
- Finalize slash menu command experience.
- Add page history/versioning primitives.
- Add a trash/archive flow for deleted notes.

## Search & knowledge

- Add semantic search and space-level retrieval (RAG).
- Provide read-it-later import for URLs with annotation support.

## Content features

- Math/LaTeX support.
- Expand table capabilities.
- Improve link interactions and editing ergonomics.

## Platform

- Validate mobile-specific context menu behavior and input focus edge cases.
